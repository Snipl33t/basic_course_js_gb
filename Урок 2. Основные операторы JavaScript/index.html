Задание 1:

Дан код:

var a = 1, b = 1, c, d;

c = ++a; alert(c); // 2

c = ++a; alert(c); // 2

d = b++; alert(d); // 1

c = (2+ ++a); alert(c); // 5

d = (2+ b++); alert(d); // 4

alert(a); // 3

alert(b); // 3

Почему код даёт именно такие результаты?

Ответ: ++a сначала инкрементирует а, а затем производятся остальные операции. b++ сначала выполняются все операции с b, а по завершению b инкрементруется.
